2. BULLET POINTS (5 POINTS)
--------------------------------------------------
[STRUCTURE]
1. WHAT IT IS: Core benefit + main use case.
2. WHAT'S INCLUDED: Pack count + specific components.

4. HOW TO USE: Installation or usage steps.

[STYLE]
- [cite_start]Start with a capitalized header (e.g., "EASY TO INSTALL —"). [cite: 34]
- [cite_start]Focus on FACTS (dimensions, materials) over fluff (amazing, perfect). [cite: 26]

[TEXT_ON_IMAGES]
- OK: Sizes, material names, "What's in the box".
- [cite_start]BAN: "Best Seller", "Amazon's Choice", Price, Promotion. [cite: 36]

4. BACKEND SEARCH TERMS
--------------------------------------------------
- [cite_start]Length: Under 249 bytes. [cite: 37]

- [cite_start]Do NOT use "Official", "Genuine", "Authorized", or "OEM" unless you have explicit authorization. [cite: 16]

2. SAFE COMPATIBILITY TEMPLATES
--------------------------------------------------
[RULE] You may use a brand name ONLY to indicate compatibility.
[SAFE_FORMAT]
- "Compatible with [Brand] [Model]"
- "Replacement for [Brand] [Model]"
- "Fits [Brand] [Model]"
[UNSAFE_FORMAT]
- "[Brand] Case" (e.g., "iPhone Case" is risky; use "Case for iPhone")

--------------------------------------------------
[HARD_BAN - ACTION WORDS]
[cite_start]diagnose, cure, treat, mitigate, prevent, heal, therapy, remedy. [cite: 22]
[HARD_BAN - CONDITIONS]

[HARD_BAN]
antimicrobial, antibacterial, antiviral, disinfectant, sanitizer, kills 99.9%, germ-free, anti-mold.
[RISK]
Using these words triggers "Pesticide" classification requiring EPA registration. [cite_start]Avoid unless registered. [cite: 22]

[SAFE_ALTERNATIVE]

--------------------------------------------------

[cite_start][USE] Water-resistant, durable, long-lasting. [cite: 23]

Purpose: Rules for Reviews, Child Safety, and Restricted Items.

1. REVIEW MANIPULATION (ZERO TOLERANCE)
--------------------------------------------------
[HARD_BAN]
- Asking for a "5-star review".
- Offering "Free gift for review".
- "Contact us before leaving a review" (Interfering with negative reviews).
- [cite_start]Inserts/Cards in packaging asking for positive feedback. [cite: 40]

2. CHILD SAFETY & WARNINGS
--------------------------------------------------
[BALLOONS]
[cite_start]Mandatory Warning: "WARNING: CHOKING HAZARD – Children under 8 yrs. can choke or suffocate on uninflated or broken balloons. Adult supervision required." [cite: 41]
[SMALL PARTS]
[cite_start]Mandatory Warning (for toys/small items): "WARNING: CHOKING HAZARD – Small parts. Not for children under 3 yrs." [cite: 42]

3. RESTRICTED ITEM CLAIMS
--------------------------------------------------
[cite_start][LASERS] No "High power", "Military grade", or "Burn" claims. [cite: 43]
[BATTERIES] Must accurately state chemistry (Lithium ion, Alkaline). [cite_start]No false longevity claims. [cite: 43]
[PLANTS/SEEDS] Strictly regulated. [cite_start]Do not use "seeds" to describe non-agricultural items. [cite: 43]
