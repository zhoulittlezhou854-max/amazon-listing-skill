# tools/ INDEX

## 目的
存放工具函数、数据加载器和预处理模块。

## 内容结构
```
tools/
├── INDEX.md (此文件)
├── __init__.py        # 包初始化
├── country_vocab.py   # 国家词汇表
├── data_loader.py     # 数据加载器
└── preprocess.py      # 数据预处理
```

## 模块说明
| 模块 | 说明 | 大小 | 最后更新 |
|------|------|------|----------|
| __init__.py | 包初始化文件 | 小 | 2026-04-03 |
| country_vocab.py | 国家特定词汇和规则 | 中 | 2026-04-03 |
| data_loader.py | 加载CSV/JSON数据 | 中 | 2026-04-03 |
| preprocess.py | 数据清洗和转换 | 中 | 2026-04-03 |

## 功能说明

### data_loader.py
- 加载CSV、JSON、Excel文件
- 统一数据格式
- 错误处理和验证

### preprocess.py
- 数据清洗和标准化
- 文本预处理
- 特征提取

### country_vocab.py
- 国家特定词汇表
- 语言规则
- 本地化处理

## 使用指南
1. 数据相关工具放入此目录
2. 保持函数单一职责
3. 添加详细文档字符串
4. 与业务逻辑分离

## 依赖关系
- 依赖 `data/` 目录的输入数据
- 被 `modules/` 目录的模块调用
- 独立于业务逻辑，可复用
