# products/ INDEX

## 目的
存放产品特定的配置文件和数据。

## 内容结构
```
products/
├── INDEX.md (此文件)
├── T70M_pro/          # T70M产品配置
├── EN_sample/         # 英语样例
├── ES_sample/         # 西班牙语样例
├── FR_sample/         # 法语样例
└── IT_sample/         # 意大利语样例
```

## 目录说明
| 目录 | 说明 | 文件数量 | 用途 |
|------|------|----------|------|
| T70M_pro | T70M产品配置 | 多文件 | 主要产品配置 |
| EN_sample | 英语样例配置 | 多文件 | 英语市场样例 |
| ES_sample | 西班牙语样例 | 多文件 | 西班牙市场样例 |
| FR_sample | 法语样例 | 多文件 | 法国市场样例 |
| IT_sample | 意大利语样例 | 多文件 | 意大利市场样例 |

## 配置类型
每个产品目录可能包含：
- 产品规格文档
- 关键词列表
- 图片资源
- 模板文件
- 运行配置

## 使用指南
1. 新产品创建独立目录
2. 目录命名：`<产品代码>_<描述>`
3. 保持配置一致性
4. 旧配置归档到 `archive/`
